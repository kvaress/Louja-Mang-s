# 📚 Loja de Mangás — Projeto PHP com XAMPP

## 📖 Sobre o Projeto

Este projeto consiste em um site de uma loja de mangás desenvolvido em **PHP** utilizando o **XAMPP** como ambiente local.
O sistema foi criado como atividade acadêmica com o objetivo de desenvolver um site completo contendo:

* Interface web funcional
* Integração com banco de dados
* Operações CRUD
* Organização de páginas e sistema
* Simulação de uma loja virtual de mangás

O projeto permitirá o gerenciamento de produtos (mangás), possibilitando cadastrar, visualizar, editar e remover informações diretamente do banco de dados.

---

# 🚀 Tecnologias Utilizadas

* PHP
* HTML5
* CSS3
* JavaScript
* MySQL
* XAMPP

---

# ⚙️ Funcionalidades

## 📦 CRUD de Mangás

O sistema possui funcionalidades completas de CRUD:

* ➕ Cadastrar mangás
* 📖 Listar mangás
* ✏️ Editar informações
* ❌ Excluir registros

---

# 🗄️ Banco de Dados

O projeto utilizará conexão com banco de dados MySQL para armazenamento das informações dos mangás.

Exemplos de dados armazenados:

* Nome do mangá
* Autor
* Gênero
* Preço
* Quantidade em estoque
* Imagem
* Descrição

## 1️⃣ Instalar o XAMPP

[XAMPP Official Website](https://www.apachefriends.org/index.html?utm_source=chatgpt.com)

---

## 2️⃣ Clonar o Projeto

```bash id="w9dk13"
git clone https://github.com/seu-usuario/seu-repositorio.git
```

---

## 3️⃣ Mover para a pasta htdocs

Coloque a pasta do projeto dentro de:

```bash id="k0fj24"
C:\xampp\htdocs\
```

---

## 4️⃣ Iniciar os Serviços

No painel do XAMPP, inicie:

* Apache
* MySQL

---

## 5️⃣ Configurar Banco de Dados

1. Abra o phpMyAdmin
2. Crie um banco de dados
3. Importe o arquivo `.sql` do projeto

Acesse:

```text id="96g0oe"

```

## 6️⃣ Executar o Projeto
